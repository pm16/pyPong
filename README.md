# pyPong
Pong using Pyglet in Python
## Setup
01. Clone repo
02. Enter cloned repo directory
03. Create a venv
```uv venv```
04. Activate venv
```souce .venv/bin/activate```
05. Install Pyglet
```uv pip install pyglet```
